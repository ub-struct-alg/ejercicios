\question
Determina los factores invariantes y los divisores elementales de los grupos 
abelianos definidos por los siguientes generadores y relaciones:

\begin{parts}
\part Generadores $a,b,c,d$. 
Relaciones $\begin{cases}
2a + 4b = 0 \\
3b = 0
\end{cases}$.
\part Generadores $a,b,c,d,e$. 
Relaciones $\begin{cases}
a - 7b - 21c + 14d = 0 \\
5a - 7b - 2c + 10d - 15e = 0 \\
3a - 3b - 2c + 6d - 9e = 0 \\
a - b + 2d - 3e = 0
\end{cases}$. 
\end{parts}
